# C. K. Ogden 的 Basic English 使用规则与技巧

> 来源：Ogden 的 *Basic English: A General Introduction with Rules and Grammar* (1930)、*The ABC of Basic English*、*The Basic Words* 等著作，整理自 zbenglish.net 网站资料。

---

## 一、总览：850 词构成

Basic English 的 850 个词分为三大类：

| 类别 | 数量 | 说明 |
|------|------|------|
| **操作词 (Operations)** | 100 | 含 18 个核心动词 (operators)、介词 (directives)、代词、连词、副词等结构词 |
| **事物名词 (Things)** | 600 | 其中 400 个普通名词 + 200 个可画名词 (picturable words) |
| **性质词 (Qualities)** | 150 | 其中 100 个一般形容词 + 50 个反义形容词 |

**核心原则**：这 850 个词能完成标准英语中约 20,000 个词的工作量。

---

## 二、核心技巧：去动词化 (Elimination of Verbs)

### 2.1 为什么去动词

Ogden 说："能用 850 词做这么多事的首要原因，就是 Basic 彻底摆脱了'动词'的束缚。"

传统的"动词"是对"操作 + 方向 + 事物"的简写 (shorthand)。例如：
- *enter* = **go into**
- *disembark* = **get off a ship**
- *insert* = **put in**
- *prepare* = **get ready**
- *hurt* = **give pain**
- *I can* = **I am able**
- *difficult* = **hard**
- *shape* = **form**

### 2.2 16 个运算符 (Operators)

Basic 只有 16 个核心动词（运算符），所有动作通过它们的组合完成：

| 动词 | 说明 |
|------|------|
| **be** | 存在、是 |
| **have** | 有 |
| **do** | 做（泛指活动） |
| **seem** | 似乎（be 的镜像） |
| **come** | 来 |
| **go** | 去 |
| **put** | 放 |
| **take** | 拿 |
| **give** | 给 |
| **get** | 得到 |
| **keep** | 保持 |
| **let** | 允许 |
| **make** | 制造（创造性变化） |
| **send** | 送 |
| **see** | 看 |
| **say** | 说 |

外加 2 个助动词：**may**（可能）、**will**（将来）

### 2.3 运算符的变位 (Conjugation)

所有运算符按正常英语规则变位：

| 原形 | 现在式 (he/she) | 过去式 | -ing 形式 | 过去分词 |
|------|----------------|--------|-----------|----------|
| come | comes | came | coming | come |
| get | gets | got | getting | got |
| give | gives | gave | giving | given |
| go | goes | went | going | gone |
| keep | keeps | kept | keeping | kept |
| let | lets | let | letting | let |
| make | makes | made | making | made |
| put | puts | put | putting | put |
| seem | seems | seemed | seeming | seemed |
| take | takes | took | taking | taken |
| be | am/is/are | was/were | being | been |
| do | does | did | doing | done |
| have | has | had | having | had |
| say | says | said | saying | said |
| see | sees | saw | seeing | seen |
| send | sends | sent | sending | sent |

### 2.4 使用名词替代动词

60 多个词形式上像动词，实际按名词使用 —— 结构为 **"make/give/have/take + 名词"** 替代动词：

- *I attempted to come* → **I made an attempt to come**
- *He attacked the town* → **He made an attack on the town**
- *She answered the letter* → **She gave an answer to the letter**

常用此类名词：
act, attack, attempt, bite, blow, burn, burst, control, cook, cry, damage, desire, doubt, drink, fly, fold, hate, hope, jump, kick, kiss, laugh, lead, love, measure, pull, push, regret, reward, roll, run, sleep, slip, smile, sneeze, start, station, stop, support, swim, taste, touch, walk, wash, whistle

### 2.5 运算符 + 方向词 = 数千个复杂概念

10 个运算符 + 20 个方向词 (directives) 组合，可生成约 200 个简单英语动词的等价表达。而在不同语境下，一个组合可以替代多个动词：

- **put in** = insert, interject, render, infuse, fold, file, plant, immerse, install 等
- **get off** = disembark, remove, descend from 等
- **go up** = ascend, increase, rise 等
- **come in** = enter, arrive 等
- **take away** = remove, subtract, confiscate 等
- **give up** = surrender, abandon, quit 等
- **go on** = continue, proceed, persist 等
- **put out** = extinguish, publish, expel 等

## 三、名词规则

### 3.1 复数形式

**基本规则**：加 `-s`

**例外**：
1. 不规则名词：**feet, teeth, men, women, knives, leaves, selves**
2. 以 `y` 结尾（前为辅音）：改为 `-ies`（如 fly → flies）
3. 以 `s, x, sh, ch, o` 结尾：加 `-es`（如 box → boxes）
4. **scissors, trousers** 只有复数形式；**sheep** 单复数同形；**news** 复数形式但作单数使用

### 3.2 复合词 (Compounds)

两个名词或名词 + 方向词可复合，符合标准英语习惯：
- birthday, copyright, milkman, newspaper
- outhouse, overland, raincoat, sundown
- coal-mine, walking-stick, jumping-place

**注意**：部分复合词虽未出现在 850 词表中但被认可：**away, become, cupboard, income, inside, today, upright, without**

### 3.3 派生词 (Derivatives)

300 个名词可派生四种形式：

| 后缀 | 词性 | 含义 | 示例 |
|------|------|------|------|
| **-er** | 名词 | 做该动作的人/物 | designer, builder, waiter |
| **-ing** | 名词 | 动作本身 | building, printing, walking |
| **-ing** | 形容词 | 主动意义（现在分词感） | moving train, loving father |
| **-ed** | 形容词 | 被动意义（过去分词感） | printed book, folded paper |

**重要**：`-ed` 结尾只是形容词，**不能**用作过去时态。不要说 "I have printed the book"（这涉及完整的动词技巧），要说 "I have the printed book"。

**区分两个 -ing**：作名词时（walking-stick）加连字符以避免混淆；作形容词时（the moving train）不加。

**注意**：部分派生词在标准英语中不常用（如 *judger, guider* 冗余；*requesting, viewing* 很少出现在名词前），但 Basic 学习者可自由使用。

### 3.4 名词作其他词性

**可作副词**：**back**（与 forward 相反的方向）

**可作形容词**：**light**（表示 pale）

**可作介词**：**round**

**可作名词的形容词**（直接作名词用）：
acid, chief, chemical, cold, complex, cut, elastic, equal, flat, female, first, flat, future, goods, hollow, last, living, male, material, opposite, parallel, past, present, public, quiet, right, safe, same, second, secret, solid, sweet, waiting, wet, wrong, young 以及所有颜色词

### 3.5 名词扩展方式

1. **延伸 (Extension)**：用表示某事物的符号指代相关事物
   - 部分代整体：letter（字母）→ letter（书信）
   - 因果：bite（咬的动作）→ bite（咬下的东西）
   - 行为者→行为：lift（举起）→ lift（电梯）

2. **特化 (Specialization)**：泛化词在特定上下文中的特定含义
   - "send in an account" → account = 账单（而非"账户"）
   - "death of a famous Judge" → judge = 法官（而非鉴赏家）

3. **隐喻 (Metaphor)**：将一种关系模式应用到另一领域
   - "the grip of a disease"（疾病的控制力）
   - "designs for the future"（未来的计划）

## 四、形容词 (Qualifiers) 规则

### 4.1 比较级和最高级

**基本规则**：在形容词前加 **more** 和 **most**

**例外**：
- good → better → best
- bad → worse → worst

**单音节形容词**：在标准英语中常用 `-er` / `-est` 后缀（Basic 学习者应适应这种用法）

**不可比较的形容词**：first, second, last, past, present, future, right, left, male, female, cut, same 等。

**例外规则**：
- 单音节词一般用 `-er` / `-est`，多音节词用 more / most
- **bent, like, wrong** 用 more / most
- **early** 用 `-er` / `-est`

### 4.2 副词构成

**基本规则**：形容词加 `-ly`

**例外**：
- 以 `-ing` 结尾的形容词（boiling, hanging, living, waiting）—— 不加 -ly
- **good**（用 **well**）
- **cut, like, awake, same, short, shut, small, tall** —— 不需要副词形式
- **female, ill** —— 因语音和谐原因不加

**拼写例外**：
- 以 `y` 结尾：变 y 为 i（happy → happily）
- 以 `ble` 结尾：去 e（simple → simply；true → truly）
- 以 `tic` / `tric` 结尾：加 `-ally`

**时间/地点副词**：不加 -ly（用原词）
**方向副词**：与形容词同形（high, low, left, right, straight, parallel）

### 4.3 否定前缀 un-

50 个形容词可加前缀 `un-` 构成否定（与反义词表中的许多词重叠）：

able, automatic, beautiful, bent, broken, certain, chemical, clean, clear, common, complete, complex, conscious, cut, elastic, electric, equal, fertile, fixed, free, frequent, happy, healthy, important, kind, like, married, medical, military, mixed, natural, necessary, normal, open, parallel, physical, political, probable, quiet, ready, regular, responsible, safe, smooth, solid, straight, sweet, tired, true, wise

**注意**：部分如 `unregular`, `unprobable` 不符合标准英语，初学者可用 `not` 回避。标准英语中这些词用 `im-`/`ir-`（如 improbable）—— 但在 Basic 中都可接受。

### 4.4 形容词派生

6 个形容词可加 `-er` 和 `-ing`（不能加 `-ed`）：
**clean, cut, dry, open, separate, shut**

### 4.5 集合名词

形容词前加 `the` 可构成集合名词：the rich, the poor, the young

## 五、介词/方向词 (Directives)

### 5.1 20 个基本方向词

Ogden 称之为"方向词" (directives)，每个都有清晰的空间定义：

| 方向词 | 根义 |
|--------|------|
| **about** | 周围 |
| **across** | 横过 |
| **after** | 之后 |
| **against** | 反对 |
| **among** | 之中 |
| **at** | 在（某点） |
| **before** | 之前 |
| **between** | 之间 |
| **by** | 在旁 |
| **down** | 向下 |
| **from** | 从 |
| **in** | 在内 |
| **off** | 离开 |
| **on** | 在上 |
| **over** | 在上方/覆盖 |
| **through** | 穿过 |
| **to** | 到 |
| **under** | 在下方 |
| **up** | 向上 |
| **with** | 和/用 |

### 5.2 非方向介词

| 介词 | 用法 |
|------|------|
| **for** | 目的（desire for food）、代替（pro-preposition） |
| **of** | 所属/密切关系（源于 off，表示"曾在上/紧邻"） |
| **till** | 时间（"到...时候"的缩写） |

### 5.3 方向组合

方向词可组合使用：**into, upon, down from**

### 5.4 隐喻扩展

方向词可通过隐喻用于非空间关系：
- 时间类比：**Come to tea, Get ready at six, Knowledge before the event**
- 虚构类比：**Thoughts come into the mind, Get at the details**
- 其他：**Go against a friend**（对抗，源于力的对立）、**Painted by Leonardo**（工具性，源于接近）、**Do it through a representative**（中介，源于穿越）

## 六、代词系统

### 6.1 代词变格表

| 数 | 性 | 主格（施事） | 宾格（受事） | 所有格 |
|----|----|-------------|-------------|--------|
| 单 | 全 | **I** | **me** | **my** |
| 复 | 全 | **we** | **us** | **our** |
| 单 | 阳 | **he** | **him** | **his** |
| 单 | 阴 | **she** | **her** | **her** |
| 单 | 无性 | **it** | **it** | **its** |
| 复 | 全 | **they** | **them** | **their** |
| 单/复 | 全 | **you** | **you** | **your** |

**独立所有格**：mine, yours, ours, theirs, his, hers（由 "this is my book" → "this is mine"）

**one** 的特殊用法：
1. 表示"任何人"：**One frequently comes across errors**
2. 回指前文提到的事物：**That book is a good one**

### 6.2 指示代词

| 数 | 近 | 远 |
|----|----|----|
| 单数 | **this** | **that** |
| 复数 | **these** | **those** |

### 6.3 疑问/关系代词

| 性 | 主格 | 宾格 | 所有格 |
|----|------|------|--------|
| 人 | **who** | **whom** | **whose** |
| 物/无性 | **which** | — | — |
| 全部 | **what** | — | — |
| 中性 | **that** | — | — |

### 6.4 名词所有格

可用 `'s` 替代 `of` 表示所属，风格更简洁：
- **the boy's book** = the book of the boy

## 七、语序规则

### 7.1 标准句型

**"主语 + 时间词 + 操作 + 宾语 + 方向 + 修饰"**

示例：**I will put the record on the machine now.**

1. **主语** (Subject) —— 做动作的人/物
2. **时间词/助动词** (Time word/Auxiliary) —— will, may, have 等
3. **操作词** (Operator) —— put, take, get 等
4. **受事宾语** (Object) —— 被作用的事物
5. **方向/地点** (Direction/Place)
6. **副词/修饰** (Adverb/Modifier)

### 7.2 形容词位置

- 一般放在名词前：a **good** book, the **red** car
- 当动词需要形容词补足时，放在名词后：**get table ready, keep door shut, make hole wide**
- 与 be 连用时：放在 be 后（The food is **good**）
- 多个形容词时：**a** 和 **the** 放在最前面

### 7.3 副词位置

- **一般副词**：放在句尾（I will give simple rules to the boy **slowly**）
- **程度副词**（almost, any, little, much, no, only, quite, so, very）：紧跟在被修饰词前
- **介词性副词**：紧接在它所伴随的介词前（The ship is **far from** land）
- **否定副词 not**：放在助动词和操作词之间（I **did not** go）
- **ever**：与否定词位置相同（否定时跟在否定词后）

### 7.4 疑问句语序

**主谓倒装**：助动词 + 主语 + 动作

- **Is** this soft?
- **What** is this?
- **How** did it get here?

对于一般现在时和一般过去时，引入助动词 **do**：
- **Do** you like it?
- **Did** he come?

### 7.5 从句语序

从句可放在句子中间或末尾，语序与主句相同。关系代词 **that, who, which, what** 引导从句。

## 八、助动词与时态

### 8.1 助动词

| 助动词 | 用法 | 备注 |
|--------|------|------|
| **will** | 一般将来 | 不区分 shall/will |
| **would** | 条件将来 | = 过去将来；**不**用 should |
| **may** | 可能性/允许 | |
| **might** | 过去的可能 | may 的过去式 |

**不用 should**：
1. 应当 → 用 **It is right for you to do your best**（替代 You should do your best）
2. 计划 → 用 **I was to go**（替代 The order was that I should go）
3. 虚拟将来 → 用 **If he came**（替代 If he should come）

### 8.2 时态构成

| 时态 | 构成 |
|------|------|
| 简单现在 | 动词原形 / -s |
| 简单过去 | 过去式 |
| 将来 | **will** + 原形 |
| 现在完成 | **have/has** + 过去分词 |
| 条件 | **would** + 原形 |
| 进行体 | **be** + -ing |
| 被动 | **be** + 过去分词 |
| 可能性 | **may** + 原形 |
| 必要性 | **have** + to + 原形 |

复合时态：**will have been done** = may have been + done 等形式。

### 8.3 时态呼应

- 主句现在/将来时 → 从句可用任何适合的时态
- 主句过去时 → 从句用过去时（除非是比较：*I was more tired than you **are***；或从句陈述适用于现在/将来：*It was his view that Britain **will be** a second-rate power*）

## 九、比较结构

### 9.1 比较级

**不等比较**：形容词比较级后接 **than**
- The sun is **more bright than** the moon.
- The moon is **less bright than** the sun.

**相等比较**：形容词前后用 **as**
- A woman is **as old as** she seems.

### 9.2 比较对象

比较不一定在两个物体/动作之间，也可以是物体与某个隐含标准：
- The food was **as good as** it was said to be.

## 十、连词 (Conjunctions)

连词用于连接词、短语或句子。

**介/连兼用**：**after, before, till**

**代/连兼用**：**that, who, which, what**

**副/连兼用**：**how, when, where, why**

**并列连词**：**and, but, or, if, though, while, because**

## 十四、学习与教学方法

### 14.1 学习时间

- 掌握 850 词：约 **30-50 小时**
- 快速路径：每天 4 小时 × 1 周
- 稳妥路径：每天 2 小时 × 1 个月（1 小时单词 + 1 小时语序和 250 个习惯用法）

### 14.2 学习顺序

1. **先阅读，后口语写作** —— 先建立识别能力
2. 每天 30 词起步，熟练后可到 50-100 词/小时
3. 使用配图教材（*The Basic Way to English* 每页都有图）
4. 将 850 词表放在口袋中随时查阅

### 14.3 口语训练

- 尽早背诵若干句型和短故事
- 快速朗读，保持节奏
- **12 个轻读词**（仅在有特殊原因时才重读）：a, the, and, of, for, from, to, than, is, some, have, that

### 14.4 重音规则

**337 个多音节词的重音规则**：
- **主要规则**（覆盖 254/337 个词）：**重音在倒数第二个音节前**
  - reason, simple, example, important, education, automatic
- 其余 83 个中，61 个可归入 3 个简单规则
- 仅 22 个真正的例外，半分钟即可记住

### 14.5 句重音规则

1. 每 10 个词中至少重读一个 —— 语义最重要的词
2. 上述 12 个轻读词在发音上与后面的词连读

### 14.6 写作训练

- 用日记记录每日事件 —— 关注"事物、思想、感受"而非"单词"
- 不要逐字翻译母语 —— 这会让你关注 Basic 没有对应词的词
- 将报纸文章改写为 Basic

### 14.7 英语母语者的学习方法

1. 在 850 词表中标记自己已会用的词
2. 找出常用但不在 Basic 中的词，查明原因
3. 学习 16 个运算符与每个方向词的组合
4. 从报纸取 10 行改写为 Basic
5. 每天分析报纸文章：标出 Basic 词和非 Basic 词，对照词表纠错
6. 对容易出错的词建立个人清单（通常不超过 10-12 个）

## 十五、翻译技巧

### 15.1 翻译本质

将标准英语翻译为 Basic 是**纵向翻译** —— 在更简单的层面重新表述，而非逐词对应。

### 15.2 关键策略

1. 理解整体思想，而非逐词翻译
2. 必要时完全重组句子结构
3. 识别"简写"词并展开：
   - 动词 → 操作词 + 方向词/名词
   - 抽象名词 → 更基础的表述
   - 情感化语言 → 事实性语言
4. 使用 *The Basic Dictionary*（含 25,000 词条）查找标准词在 Basic 中的对应表达

### 15.3 示例

> 原文：*All will in the end give up the use of force.*
> Basic：**They believe all of the nations of the world must come to the abandonment of the use of force.**

## 十六、写作中的常见问题与建议

### 16.1 语气词的处理

- 不要使用 "Well, ..."、"You see, ..." 等填充词
- 不要使用俚语（必须使用时加引号）
- 避免文学性的修辞和隐喻

### 16.2 避免的词法问题

- 不要把名词当动词用（Nouns must not be used as verbs）
- 不要把 -ed 当作过去时（它是形容词）
- 不要把介词放在没有意义的位置

### 16.3 介词可以放句尾

Basic 允许介词放在句尾，这是标准英语的悠久传统：
- He is getting **on**.
- This is the horse which he is getting **on**.

### 16.4 口头缩写

口语中常用缩写：**I'll** (= I will), **I'd** (= I would), **don't** (= do not)

## 十八、关键理念摘要

1. **850 词，不是 851** —— 先完全掌握 850 词，再考虑添加任何新词
2. **动词是多余的** —— 用"操作词 + 名词/介词"替代
3. **先阅读，后写作** —— 建立理解力后再输出
4. **关注事物和关系，而非单词** —— Basic 不是逐词翻译
5. **清晰胜于风格** —— Basic 的目的是精确传达信息，而非文学修饰
6. **循序渐进** —— 从根义到引申义，从规则形式到习惯用法
7. **词频≠价值** —— 选择标准是"能做什么"，而非"多常用"
8. **处处可学** —— 广告牌、报纸、广播新闻都是练习素材
